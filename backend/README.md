# Hilo Backend

This is the backend service for the Hilo application. It is built with **Node.js** and **Express.js** and provides APIs for user authentication, post management, and profile management. The backend uses **MongoDB** as the database and integrates with **Cloudinary** for image uploads.

## Features

- **User Authentication**: Signup, login, and logout functionality using JWT.
- **Posts**: APIs to create, read, like/unlike, delete, and reply to posts.
- **Profile Management**: APIs to fetch and update user profiles.
- **Follow/Unfollow**: API to follow and unfollow users.
- **Feed**: Fetch posts from followed users.

## Tech Stack

- **Framework**: Express.js
- **Database**: MongoDB (using Mongoose for object modeling)
- **Authentication**: JWT (JSON Web Tokens)
- **Image Uploads**: Cloudinary
- **Environment Variables**: dotenv
